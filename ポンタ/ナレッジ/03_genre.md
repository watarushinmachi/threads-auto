# ポンタ ジャンル設計

## どんなジャンルか
- メインジャンル：Threads × Claude Code で稼ぐ方法
- サブジャンル：AI活用、借金返済ドキュメンタリー

## 投稿配分
- AI活用術（Claude Code中心）：5割
- 借金返済リアル報告：3割
- 実績報告（ルナの成果含む）：2割

## ジャンルの選定理由
- AI系：低単価商品販売（1〜10万円）に適したジャンル
- 「Threads × Claude Code」は競合がほぼゼロのブルーオーシャン
- ドキュメンタリー要素：一次情報として価値が高い
- 借金返済：共感を呼びやすく、ファン化しやすい

## コンセプト
- 誰の：Threadsで稼ぎたいがやり方がわからない人の
- どんな悩み：投稿作成が大変、何を書けばいいかわからない、時間がない
- どんな手段で：Claude Codeを使ったThreads運用の全自動化で
- どんな未来：AIに任せてThreadsで月5万〜30万稼ぐ仕組みを作る

## ゴール商品
- **「Threads × Claude Code で稼ぐ完全ガイド」（note 5,000円）**
- 目標：100冊販売 = 50万円
- 内容イメージ：
  - Claude Codeの基本設定
  - ナレッジベースの作り方（8ファイル構成）
  - 6つのAIエージェントスキルの構築方法
  - 投稿自動生成→Discord承認→自動投稿の仕組み
  - ルナアカウントでの実証結果
- ポンタ自身の運用過程がそのまま教材のネタになる
