# AI活用術 専門知識

## ポンタが発信するAI知識の範囲
- **Claude Code** を使ったThreads運用の全自動化（メイン）
- ChatGPT / Claude の活用法
- AIを使った記事作成・コンテンツ作成
- AIツールの比較・レビュー（一次情報として）
- AI副業の具体的な手順・収益報告

## Claude Code × Threads が核心
- 6つのAIエージェント（リサーチャー、アナリスト、ライター、ポスター、フェッチャー、スーパーバイザー）
- ナレッジベースの8ファイル構成
- Discord連携（承認フロー、自動通知）
- Threads API連携（自動投稿、データ取得）
- これらの仕組みを作る過程自体がコンテンツ

## 機能的価値を最大化するポイント
- 具体的なノウハウ、手段、ツールなど実用的な情報を提供
- Threadsでは機能的価値を含む投稿が少ない（希少性が高い）
- 「Claude Codeのプロンプト」「具体的な設定手順」など再現性の高い情報
- フォロワー増加率が高い（10万表示で1,000人獲得も可能）

## 情緒的価値との使い分け
- 機能的価値（AI活用術）：5割 → フォロワー獲得の主力
- 情緒的価値（借金返済の自己開示）：3割 → ファン化の主力
- 実績報告：2割 → 権威性の構築
