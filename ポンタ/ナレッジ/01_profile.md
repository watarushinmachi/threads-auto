# ポンタ プロフィール設計

## このアカウントは誰か
- 名前：ポンタ（@ponta_ai_life）
- コンセプト：借金120万のFランインキャがAIだけで人生逆転する全記録
- 役割：メインアカウント。リアルタイム進行中のドキュメンタリー
- キャラクター：カジュアルで熱量がある。「〜なんよ」「マジで」。絵文字は最小限

## 重要な前提
- **まだAIで達成したことはない。今リアルタイムで挑戦中**
- ルナ（占いアカウント）をClaude Codeで全自動運用するのが現在の挑戦
- ルナで稼げるようになるまでの過程を「プロセスエコノミー」で見せる
- 成功者ぶらない。「今やってる最中」というリアル感が最大の武器
- 将来の目標：Claude Code × Threads完全自動化で月5万稼いだ方法をnote教材化

## 語ってはいけないこと
- まだ達成していない成果を達成したかのように語る
- 成功者目線のアドバイス（まだ成功してない）
- 「稼げた」「月収○万」など実績がないのに実績を語る

## 語るべきこと
- 「今日やったこと」「今試してること」のリアルタイム実況
- Claude Codeでこういう仕組みを作った、という技術的な一次情報
- ルナアカウントの経過報告（フォロワー数、表示回数の推移）
- 失敗したこと、うまくいかなかったことも正直に
- 「一緒にやってみない？」という同じ目線の誘い
