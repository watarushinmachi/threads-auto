# ポンタ 戦略

## ゴール
- **note「Threads × Claude Code で稼ぐ完全ガイド」5,000円 × 100冊 = 50万円**
- ルナアカウントの実証結果を「証拠」として使う

## 現在のフェーズ
- 0〜100フォロワー：とにかくバズらせ続ける
- 1日5〜10投稿、完璧を求めず数をこなす

## マネタイズの3段階
1. **集客**（投稿をバズらせる）→ フォロワー1,000人目標
2. **教育・ファン化** → Claude Codeの可能性を見せる
3. **マネタイズ** → note販売（5,000円）

## 逆算ロードマップ

### Phase 1：集客＋実績作り（〜フォロワー500人）
- 毎日10投稿で認知拡大
- ルナアカウントをClaude Codeで全自動運用→実績を積む
- 「借金120万だけどAIで自動運用アカウント作ったらフォロワー〇〇人になった」の一次情報
- アフィリエイト（AI系ツール、A8.net等）も並行

### Phase 2：教育（フォロワー500〜1,000人）
- 6つの教育（信頼・目的・問題・手段・行動・投資）を投稿に散りばめる
- 信頼の教育：ルナの実績、自分の運用過程を見せる
- 問題の教育：「手動で毎日10投稿、きつくない？」
- 手段の教育：「Claude Codeなら全自動化できる」
- note制作の過程をリアルタイムで共有（プロダクトローンチ）

### Phase 3：ローンチ（フォロワー1,000人〜）
- 4週間前：匂わせ開始
- 2週間前：教育・進捗共有・強み提示・カウントダウン
- 当日：販売開始
- 価格設計：初回限定500円→通常5,000円

## 集客導線
1. Threads投稿でバズ→フォロワー獲得
2. プロフィールリンクからnote
3. note販売（5,000円）

## ドキュメンタリー戦略（ポンタ独自）
- ルナアカウントでの成果をポンタで報告
- 「AIだけで運用したアカウントがフォロワー〇〇人達成」
- 「Claude Codeでこの仕組み作るのに使った時間は〇時間だけ」
- 自分の挑戦の過程自体がコンテンツ
- noteの中身＝ポンタが実際にやったことそのもの

## 無料→有料の流れ
1. 最初の1ヶ月は有益情報を無料で出しまくる
2. 「この人の情報すごいのに全部タダ？」と思わせる
3. 「全部まとめたnote出します」→ 売れる

## 逆算設計の原則
- 「フォロワーを増やしてからマネタイズ」ではない
- 購買意欲の高いユーザーを集められるアカウントが高収益
- フォローされた瞬間から収益化できる動線を最初に作る
