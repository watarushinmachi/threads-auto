# ルナ 戦略

## ゴール
- **月5万円の収益達成（3〜6ヶ月目標）**

## 運用方針
- AI全自動生成・全自動投稿（Claude Code + Threads API）
- 毎日5投稿（7:00/10:00/13:00/18:00/21:00）
- ポンタのドキュメンタリーの「実験台」としての役割も

## マネタイズ戦略：ココナラ + note の二本柱

### 根拠（リサーチ結果）
- 占い系アフィリエイトは1成約3,000〜8,000円だがフォロワー5,000人以上が目安
- ココナラは出品初日からOK、プラットフォーム側に集客力がある
- noteは一度書けばストック型収入になる
- **この2つの併用が最短で月5万円に到達するパターン**

### 収益シミュレーション
| 収益源 | 単価 | 月間件数 | 月収 |
|--------|------|---------|------|
| ココナラ テキスト鑑定 | 2,000円（手取り1,200円） | 20件 | 24,000円 |
| note 有料記事 | 1,000円（手取り850円） | 30部 | 25,500円 |
| **合計** | | | **49,500円** |

### 中期的に追加
- 占いアフィリエイト（フォロワー増加後）：1成約5,000円×月5件 = 25,000円
- LINE公式→個別鑑定：3,000円×月10件 = 30,000円

## 逆算ロードマップ

### Phase 1：集客＋無料実績（月1〜2）
- Threads毎日5投稿を継続（全自動）
- 12星座別まとめ投稿でバズ狙い
- コメント誘発（「星座を教えて→個別占い」）
- **無料鑑定を10〜20件実施して口コミ・実績を蓄積**
- ココナラアカウント作成、最初は500円の低価格で出品

### Phase 2：有料化テスト（月2〜3）
- ココナラの価格を段階的に引き上げ（500円→1,000円→2,000円）
- noteに有料記事を公開（「星座別 恋愛運完全ガイド」500〜1,000円）
- Threads→ココナラ/noteへの導線をプロフィールに設置
- LINE公式アカウント開設、プロフィールから誘導開始

### Phase 3：月5万円達成（月3〜6）
- ココナラでリピーター確保
- noteを月1〜2本追加（星座別、シーズン別）
- 教育投稿を散りばめる（問題の教育→手段の教育）
- 占いアフィリエイトも導入検討

## 導線設計

```
Threads投稿（認知拡大・バズ）
  ↓ コメントで星座を聞く
  ↓ 無料の簡易占い返信（ファン化）
  ↓ プロフィールリンク
LINE公式 or ココナラ or note
  ↓
有料鑑定 / note購入
```

## 商品ラインナップ（段階的）

### 入口商品（無料〜500円）
- Threads上での無料星座占い（コメント返信）
- note 500円「今月の12星座恋愛運」

### 中間商品（1,000〜2,000円）
- note 1,000円「星座別 恋愛完全攻略ガイド」
- ココナラ 2,000円 テキスト個別鑑定

### 上位商品（3,000〜5,000円）
- ココナラ 3,000〜5,000円 詳細個別鑑定＋恋愛相談
- LINE限定 月額1,000円サブスク（中期的に検討）

## KPI
- 毎日5投稿の継続率
- コメント数の推移
- フォロワー増加率
- 投稿あたりの平均表示回数
- ココナラ月間受注数
- note月間販売部数
- **月間収益（目標5万円）**

## 成功のポイント（リサーチから）
- マネタイズできないのはフォロワー数の問題ではなく「教育不足」と「導線設計の欠如」が原因
- Threadsは占い系と相性が良い（テキストベース、女性56%、ブルーオーシャン）
- 副業占い師として月5万円（年60万円）は現実的な初期目標
- AI活用で運用を効率化すれば1日1時間程度で回せる
