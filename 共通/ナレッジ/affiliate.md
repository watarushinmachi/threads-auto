# アフィリエイト攻略

## 全体像
- ASP（アフィリエイトサービスプロバイダー）への登録が必須
- 効果的に実践すれば月50〜100万円の収入も可能

## ステップ
1. 自分のジャンルに合った案件選び
2. フォロワー数の増加（まずは100人が目標）
3. 投稿とアフィリエイトリンクの適切な配置（**本文ではなくコメント欄に**）
4. 継続的な投稿と案件の紹介

## おすすめASP一覧

| ASP | 特徴 |
|-----|------|
| **楽天アフィリエイト** | 楽天市場の全商品。どのSNSでも公認で安全。スーパーセール時期に大きな収益 |
| **A8.net** | 業界最大手。約7,000以上の広告主。セルフバック機能あり。初心者向き |
| **アクセストレード** | 金融系（クレカ、FX、投資）に強い。成果承認スピードが早い。ボーナス制度あり |
| **TCS** | テクノロジー系が豊富。高額報酬案件多い |
| **afb（アフィビー）** | 美容・健康商品が充実。女性向け。審査基準が比較的緩やか |
| **Smart-C** | BtoB向け・アプリ系。法人向けサービスの報酬率が高い |
| **もしもアフィリエイト** | W報酬プログラム（最大12%上乗せ）。Amazon商品を扱える数少ないASP |

## Threadsでのアフィリエイト注意点
- **本文にリンクを貼ると表示回数が大幅減少**
- リンクはコメント欄に貼る
- 「稼げる」系の訴求はシャドウバンのリスク
- 情報商材関連のリンクを頻繁に貼ると規制される可能性
- 楽天アフィリエイトはどのSNSでも公認されており安全
- 高単価案件（5,000円以上/件）があるASPを選ぶのが効率的

## 初心者の進め方
- まず手動でアフィリエイト投稿を3投稿作成（スキルを養う）
- GPTsも活用できるが、まずは手動で作成して投稿作成スキルを養う
